플렉스레이아웃

부모자인 컨테이너 박스를 통해 자손자인 아이템박스를 자유배치한다.

  • container (부모속성) 언어
    • display : flex 선언문
    • flex-direction : row / column 으로 가로정렬 / 세로정렬 표기
      row(기본값) , reverse 를 통해 역방향가능
    • flex-wrap: wrap/nowrap
      item의 가로값을 인정해주기위해 줄바꿈으로 표현
      nowrap(기본값)
    • justify-content: item 정렬방법 / flex-start / flex-end / center / space-between / space-around
    • align-item : 아이템의 길이조절 또는 한줄로 있을경우 정렬방식
      stretch 라는 확장의 개념의 속성이 기본값 그외는 justify-content 와 같음
  • item (자손자속성) 언어
    • align-self : align-item 에서 사용한 속성과 같으나 건바이건 적용
    • order : item의 숫자를 설정한다. 숫자가 클수록 뒤로 밀림 기본값은 0이며음수값(마이너스) 값까지 있음
