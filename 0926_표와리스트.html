목록태그란?

내비게이션메뉴를 만들거나, 같은특징을 가지고 있는 글들을 나열할때 사용

  • ol = orderlist의 약자 로 순서가 있는 목록제목
  • ul = unorderlist의 약자로 순서가 없는 목록제목
  • li = list의 약자로 목록의 내용을 작성
  • 특징1-들여쓰기가 사용됨(일정크기의 margin과 padding이 적용됨)
  • 특징2-앞에 기호혹은 숫자가 나타남
  • 특징3- ol,ul,li 모두 블럭태그임
  • 특징4 - li에 목록의 내용을 작성하며 내부에 ol,ul을 중복할수 있다.

테이블태그란?

html을 활용하여 도표를 제작할때 사용하는 태그
도표는 table 태그로 제작
가로공간은 tr 태그로 제작
분할공간은 td 태그로 제작하고 내용을 작성한다.

  • 가로병합 -colspan ="갯수"
  • 세로병합 -rowspan ="갯수"
  • 주의사항 td태그 속성에 사용
