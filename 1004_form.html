입력양식태그

input 태그란?

다양한 폼에서 사용자가 입력한 정보를 받을때 사용하는 태그

종류 설명 예시
text 한줄짜리 텍스트입력박스
password 비밀번호입력박스 ****으로 나타남
search 검색창입력박스 X아이콘이나타남
email 이메일입력박스 입력양식과다를경우경고내용이나타남
tel 전화번호를 입력하는박스/ pattern을 설정할수 있음
checkbox 2개이상의 복수선택이 가능 국어 수학
radio 단일선택을 해야하는 동그란버튼 / name이라는 속성이 같을경우에만 적용
여성 남성
number 숫자를 조절하는 박스 다양한 속성으로 조절가능
  • min - 최소값 min아래값으로 내려가지않음
  • max - 최대값 max 위값으로 올라가지 않음
  • step - 단계값 지정된숫자만큼씩 올라감
  • placeholder - 예시값 읽을수있게 보여주는글
range 슬라이더막대바
color 컬러를 지정할수 있는 입력박스
date,month,week,time,datetime,datetime-local local 기준으로 날짜,시간등의 정보를 보여주는 박스
button 일반버튼을 만듬 , button태그와는 다름
submit 확인버튼 으로 제출할때 주로 사용
from태그 내부에서만 작동됨
reset 다시작성버튼 으로 기존내용이 지워짐 / form 태그 내부에서만 설정가능
file 파일을 첨부할수 있는 버튼

fieldset 태그

하나의 공간박스를 만드는 구조의 입력양식

legend 태그

fieldset 태그의 박스 안쪽에 작성되는 제목양식

label 태그

input태그의 설명을 작성할때 세트로 제공되는 구조형식 인라인형식으로 input 옆쪽으로 붙어서 나타남
input id = '이름' label for = '이름'

input 기타속성

autofocus 자동으로 지정해서 위치를 알려주는 구조
readonly text입력필드에 지정된글자를 읽게만 하는 구조
required 필수작성요소_내용이 빠져있을경우 확인창이 눌리지않음
