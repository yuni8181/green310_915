백그라운드속성

background :속성에 한줄에 작성가능 작성시 순서는 관계없으며 띄어쓰기로 구분함

상세속성

  • background-color : 색깔을 넣어줌
  • background-image : 배경이미지를 넣어줌 url('이미지경로주소')-따옴표생략가능
  • background-repeat : repeat / repeat-x (가로반복) / repeat-y(세로반복) no-repeat(반복없음)
  • background-position : 이미지를 넣었을경우 위치를 조정 수평(left/right/center) 수직(top/bottom/center) left top을 기준으로 수치값으로도 이동가능 2
  • background-size : 이미지의 크기 contain(이미지가 잘리지 않은 상태에서 화면을 꽉채움 여백이생길수 있음 ) cover(요소안에 배경이미지를 꽉채움 - 화질이깨지거나 이미지가 잘려나올수 있음) 크기,%로도 나타낼수 있음
  • backgroud-attachment: 배경이미지고정 scroll(올라감)/ fixed(고정됨 left/top 00 기준)
